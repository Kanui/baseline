Baseline

Overcome your fear of heights

☯ Develop an elegant "pixel perfect" layout.

O Baseline é uma biblioteca CSS desenvolvida para resolução de problemas que envolvem padrões de estilo e design. Assim como uma biblioteca de sistemas de grid funciona, o foco do Baseline em sí, são ritmos verticais e responsividade vertical. Basta condigurar o tamanho base do seu "grid vertical" e começar a experimentar.

✍ Use your favorite preprocessor (.sass / .scss).

If you love SASS/SCSS as we are, you just need to include our /src in your project and start to define your vertical rythm.

♡ If you love designers, this is your library.

One of our main goals when developing Baseline was to create standards for vertical sizes to make design-developer communication easier. If you're a designer, you can download our guided PSD with the default rythm (10 px). In case you need to use other sizes, there's a great extension called GuideGuide that allows you to set both vertical and horizontal grid as needed.
