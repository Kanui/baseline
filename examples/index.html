Examples

Typography mixin:
/* baseline.sass */

/* Set your baseline here */
$bl: 10px
/* your code */
h1
  +bl-typo(4)
p
  +bl-typo(2)
  font-style: italic
Output:
h1 {
  margin: 0;
  padding: 0;
  font-size: 40px;
  line-height: 40px;
}

p {
  margin: 0;
  padding: 0;
  font-size: 20px;
  line-height: 20px;
  font-style: italic;
}
